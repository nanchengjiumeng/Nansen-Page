## 自学导读

### What are you going to do now, George?

你现在准备干什么，乔治？

`be going to`，是打算、准备、按计划在最近做某事，表示将来。请参见本课语法部分。

### Pink's = Pink is.

### It's for my daughter, Susan.

是为我的女儿苏珊做的。

`Susan` 作 `my daughter` 的同位语。

## 语法 Grammar in use

### 将来时 be going to

（1）将来时 `be going to` 的形式由 `am/is/are going to` + 动词原形构成。

（2）将来时 `be going to` 的用法：

A. 表示"打算"、"准备"在最近做某事。（在非正式语体中，一般多用 `be going to`，而不用 `will`。）

<details class="nce-translation">
<summary>I am going to put it on the floor.</summary>
<p>我打算把它放在地板上。</p>
</details>

<details class="nce-translation">
<summary>He is going to paint the bookcase tomorrow.</summary>
<p>他准备明天给书架刷漆。</p>
</details>

B. 表示按计划、安排要发生的事：

<details class="nce-translation">
<summary>The meeting is going to begin at nine.</summary>
<p>会议将在 9 点开始。</p>
</details>

<details class="nce-translation">
<summary>Where are you going to build the road?</summary>
<p>你们将在什么地方筑路？</p>
</details>

C. 表示预言一件事即将发生：

<details class="nce-translation">
<summary>It's going to rain!</summary>
<p>天要下雨了！</p>
</details>

<details class="nce-translation">
<summary>She's going to faint!</summary>
<p>她要晕倒了！</p>
</details>

（3）`be going to` 的疑问式与否定式

将助动词 `be` 提至句首可将陈述句变为一般疑问句。如：

<details class="nce-translation">
<summary>George is going to paint it pink.</summary>
<p>乔治打算把它刷成粉红色。</p>
</details>

<details class="nce-translation">
<summary>Is George going to paint it pink?</summary>
<p>乔治打算把它刷成粉红色吗？</p>
</details>

在助动词后面加上 `not` 可以得到否定句：

<details class="nce-translation">
<summary>George is not going to paint it pink.</summary>
<p>乔治不打算把它刷成粉红色。</p>
</details>

## 课堂笔记

### work / job / task / employment / labour / occupation / profession

| 单词 | 释义 |
| --- | --- |
| work | 可做名词、动词，指代一切的劳动、各种各样的劳动 |
| job | 有报酬的一份工作 |
| task | 愿意承担的任务 |
| employment | 雇佣关系的劳动 |
| labour | 一般指体力劳动 |
| occupation | 职业，训练有素、有专长的职业 |
| profession | 脑力劳动 |

### work hard / hard work / hard-working

| 表达 | 词性 | 释义 |
| --- | --- | --- |
| work hard | 动词词组 | 努力工作 |
| hard work | 名词词组 | 艰苦的工作 |
| hard-working | 形容词 | 勤奋 |

### 现在进行时

- 表示现在正在进行或者发生的动作、状态
- 表示现阶段正在进行或发生，但说话的此刻动作不一定进行
- 后面加一个将来的时间，可以表示打算做的事情

### be going to 句型

1. 打算要做某事
2. 即将要发生某事

`to` 后接动词原形，`be` 根据人称的变化而采用不同的形式。

<summary>I'm going to make a bookcase.</summary>

</details>

<p>你打算做一个书架吗？</p>
</details>

<details class="nce-translation">
<summary>What colour are you going to paint it?</summary>
<p>你打算把它漆成什么颜色？</p>
</details>
