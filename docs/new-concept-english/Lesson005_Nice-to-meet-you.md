## 自学导读

### Good morning.

英语中常见的问候用句。对此问候的回应一般也是 `Good morning`。根据一天中见面时间的不同还可以说 `Good afternoon`（下午好）和 `Good evening`（晚上好）。有时英美人见面时只简单地说一声 `Hello`。

### This is Miss Sophie Dupont.

这是将某人介绍给他人时常用的句式：`This is + 姓名`。

课文中的例子还有：

<details class="nce-translation">
<summary>Sophie, this is Hans.</summary>
<p>索菲娅，这位是汉斯。</p>
</details>

<details class="nce-translation">
<summary>And this is Naoko.</summary>
<p>这位是直子。</p>
</details>

### Mr. Blake / Miss Sophie Dupont

英语国家中人的姓名通常由 3 部分组成，即：名 + 中间名 + 姓。

在一般情况下，不用中间名。在熟悉的人中间，以名相称，而在正式的场合中常用 `Mr.`（先生），`Mrs.`（太太），`Miss`（小姐）或 `Ms.`（女士）这些称呼再加上姓。

`Mr.` 用于男士的姓之前，不能单独使用，如课文中的 `Mr. Blake`；而 `sir` 一般单独使用，是对长者、上司或男顾客的尊称，如：

<details class="nce-translation">
<summary>Sorry, sir.</summary>
<p>对不起，先生。</p>
</details>

`Miss` 一般用于指未婚女子，不过有时在不知道对方是否已婚时也可使用。

### Nice to meet you.

用于初次与他人见面等非正式场合。对方的回应一般应为 `Nice to meet you, too`（我也很高兴见到你）。

人们在正式的场合初次见面时常用：

<details class="nce-translation">
<summary>How do you do?</summary>
<p>你好。</p>
</details>

相应的回答也是 `How do you do?`。这是一句问候语，并非问话。

### 国籍与国家名称有别

请不要将国家名称和与其对应的国籍搞混。课文的 `French`，`German`，`Japanese`，`Korean` 以及 `Chinese` 都是表示国籍的词。句中表示中国国籍的词应为 `Chinese` 而不是 `China`。

## 语法 Grammar in use

### 特殊疑问句

以疑问词 `who`，`what`，`when`，`which`，`why`，`where`，`whose`，`how` 等引导的问句都叫特殊疑问句，也叫疑问词疑问句，有时还被称为 `wh-` 问句（wh-question）。

结构一般为：疑问词 + 助动词 + 主语 + 主动词等。

作为疑问词的 `what` 可以用来询问名字、国籍、工作、颜色、型号等。例如：

<details class="nce-translation">
<summary>What make is this car?</summary>
<p>这辆小汽车是什么牌子的？</p>
</details>

<details class="nce-translation">
<summary>What nationality are you?</summary>
<p>你是哪国人？</p>
</details>

<details class="nce-translation">
<summary>What is your job?</summary>
<p>你的工作是干什么？</p>
</details>

<details class="nce-translation">
<summary>What colour is it?</summary>
<p>它是什么颜色的？</p>
</details>

<details class="nce-translation">
<summary>What size is this skirt?</summary>
<p>这条裙子是多大号的？</p>
</details>

### a 和 an

<details class="nce-translation">
<summary>Sophie is a new student.</summary>
<p>索菲娅是一名新学生。</p>
</details>

<details class="nce-translation">
<summary>This is an umbrella.</summary>
<p>这是一把雨伞。</p>
</details>

这两个句子中出现的 `a` / `an` 在英语中被称为不定冠词。`a` 和 `an` 在意义上没有区别。用 `a` / `an` 时，我们必须记住两条基本原则：

1. `a` / `an` 有不确定的意义，即所说的人、动物或东西对听者或读者来说可能是不知道的。
2. `a` / `an` 只能用于单数可数名词之前。

此外还需注意 `a` 和 `an` 的发音：`a` 用于辅音音素之前；`an` 用于元音音素之前。当我们把 `a` 或 `an` 用来说明字母表中的字母时，便会一目了然：

<details class="nce-translation">
<summary>This is a B / C / D / G / J / K / P / Q / R / T / U / V / W / Y / Z.</summary>
<p>这是一个 B / C / D / G / J / K / P / Q / R / T / U / V / W / Y / Z 字母。</p>
</details>

<details class="nce-translation">
<summary>This is an A / E / F / H / I / L / M / N / O / S / X.</summary>
<p>这是一个 A / E / F / H / I / L / M / N / O / S / X 字母。</p>
</details>

## 课堂笔记

### This is...

<details class="nce-translation">
<summary>This is Miss Sophie Dupont.</summary>
<p>这位是索菲娅·杜邦小姐。</p>
</details>

介绍他人的时候经常用 `This is...`。

### 冠词

英语中有 3 个常见冠词：

| 冠词 | 类型 | 说明 |
| --- | --- | --- |
| a | 不定冠词 | 一个；用于辅音音素前 |
| an | 不定冠词 | 一个；用于元音音素前 |
| the | 定冠词 | 表示特指 |

`a`、`an` 用于：

1. 可数名词单数前面。
2. 具有不确定意义的事物或名词前面。

<details class="nce-translation">
<summary>This is a watch.</summary>
<p>这是一块手表。</p>
</details>

`a` 用在普通名词前面，`an` 用于元音音素（元音发音的单词）前面：

| a | an |
| --- | --- |
| a pen | an apple |
| a university | an umbrella |
|  | an egg |
|  | an hour |

`the` 是定冠词，表示特指：

<details class="nce-translation">
<summary>Look, look, a man is coming into the classroom.</summary>
<p>看，一个男人正走进教室。</p>
</details>

<details class="nce-translation">
<summary>The man is my teacher.</summary>
<p>这个男人是我的老师。</p>
</details>

### and 和 too

<details class="nce-translation">
<summary>And this is Naoko.</summary>
<p>这位是直子。</p>
</details>

`and` 用在句首表示转折。

<details class="nce-translation">
<summary>She's Chinese, too.</summary>
<p>她也是中国人。</p>
</details>

`too` 一般用于肯定句中；`either` 也表示“也”，用于否定句中。
