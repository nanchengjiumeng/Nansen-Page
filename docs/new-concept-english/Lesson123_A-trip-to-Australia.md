## 自学导读

### What a beautiful ship!

多漂亮的轮船啊！`what` 引导的感叹句通常是由 `What + a(n) + 形容词 + 名词` 构成的。

### That's right

对。这句是对上面附加疑问句的回答。上面一句虽用否定形式提问，但回答则根据事实来决定是用肯定或否定形式。`That's right` 是肯定的回答，相当于 `Yes, it is`（那正是我）。

### grow a beard / shave it off

`grow a beard` 留胡子。`shave it off` 把胡子刮掉。`it` 指 `a beard`。

## 语法 Grammar in use

### 定语从句中的省略

当关系代词代表主语并且从句中的谓语动词是（现在）进行时态时，关系代词及助动词 `be` 均可省略。如果关系代词在从句中代表宾语，则往往可以省略。此外，定语从句可用介词结尾。

## 词汇学习 Word study

<p>他说，如果他有很多钱的话，他将会周游全世界。</p>

## 课堂笔记

### 定语从句中关系代词的省略

| 情况 | 可否省略 |
| --- | --- |
| 关系代词作宾语 | 通常可省略 |
| 关系代词 + be + doing | 两者都可省略 |
| 关系代词作主语 | 不可省略 |

### What 感叹句

| 结构 | 例句 |
| --- | --- |
| What + a(n) + adj. + 单数名词 | What a beautiful ship! |
| What + adj. + 复数名词 | What beautiful flowers! |
| What + adj. + 不可数名词 | What fine weather! |

### grow / shave

| 短语 | 释义 |
| --- | --- |
| grow a beard | 留胡子 |
| shave (it) off | 刮掉（胡子） |
